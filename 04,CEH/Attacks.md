## 1) Phishing Attack

**Definition:**
A phishing attack tricks users into revealing sensitive information like passwords or credit card details through fake emails or websites.

## 2) Brute Force Attack

**Definition:**
An attacker tries many passwords repeatedly until the correct one is found.

## 3) Malware Attack

**Definition:**
Malware is malicious software designed to damage systems, steal data, or gain unauthorized access.

**Common malware types:**

#### Virus

**Definition:**
A virus is a type of malicious software that attaches itself to a legitimate file or program and spreads when the infected file is executed.

**Key points**
- Needs user action to spread (opening a file/program)
- Can damage files or system data
- Replicates by infecting other files
#### Worm

**Definition:**
A worm is malware that self-replicates and spreads automatically across networks without user interaction.

**Key points**
- No need to attach to a file
- Spreads through network vulnerabilities
- Can quickly infect many computers
#### Trojan

**Definition:**
A Trojan (Trojan Horse) is malware that disguises itself as legitimate software to trick users into installing it.

**Key points**
- Looks like a normal or useful program
- Creates backdoor access for attackers
- Does not replicate itself
#### Spyware
**Definition:**
Spyware is malicious software designed to secretly monitor user activity and collect information without the user's knowledge.

**Key points**
- Tracks browsing habits
- Steals login credentials
- Records keystrokes
#### Ransomware
**Definition:**
Ransomware is malware that encrypts a victim’s files or locks their system and demands payment (ransom) to restore access.

**Key points**
- Files become inaccessible
- Victim must pay cryptocurrency
- Often spreads through phishing emails

## 4) Command and Control (C2)

**Definition:**
After infecting a system, malware connects to a Command and Control server to receive instructions.


## 5) DDoS Attack (Distributed Denial of Service)

**Definition:**
A DDoS attack floods a server or network with huge traffic to make it unavailable.


## 6) Man-in-the-Middle (MITM)

**Definition:**
An attacker secretly intercepts communication between two parties.


## 7) SQL Injection

**Definition:**
SQL Injection happens when attackers insert malicious SQL commands into input fields to access or manipulate databases.


## 8) Cross-Site Scripting (XSS)

**Definition:**
XSS allows attackers to inject malicious scripts into web pages viewed by users.

## 9) Privilege Escalation

**Definition:**
An attacker gains higher access permissions than originally allowed.


## 10) Data Exfiltration

**Definition:**
Unauthorized transfer of sensitive data outside an organization.
