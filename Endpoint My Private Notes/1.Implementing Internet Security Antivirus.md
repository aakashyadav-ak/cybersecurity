#  Importance of Internet Security

Internet Security refers to the measures and practices designed to protect computers, networks, programs, and data from unauthorized access, attacks, damage, or theft while connected to the internet.

| Reason                  | Description                                 |
| :---------------------- | :------------------------------------------ |
| **Data Protection**     | Protects sensitive personal & business data |
| **Financial Safety**    | Prevents financial fraud and theft          |
| **Privacy**             | Keeps personal information private          |
| **Business Continuity** | Prevents disruptions to operations          |
| **Reputation**          | Protects company/personal reputation        |

#### Threats We Face Online
```
1. Malware (Viruses, Worms, Trojans)
2. Phishing Attacks
3. Ransomware
4. Identity Theft
5. Data Breaches
6. Man-in-the-Middle Attacks
7. Denial of Service (DoS) Attacks
```

#  Malware
**Malware = Malicious + Software**
Any software intentionally designed to cause damage, steal data, or gain unauthorized access to computer systems.

### Types of Malware
| Type | Description | Behavior |
| :--- | :--- | :--- |
| **Virus** | Attaches to legitimate files | Needs user action to spread |
| **Worm** | Self-replicating malware | Spreads automatically via network |
| **Trojan** | Disguised as legitimate software | Creates backdoor access |
| **Ransomware** | Encrypts your files | Demands payment for decryption |
| **Spyware** | Secretly monitors activity | Steals passwords, keystrokes |
| **Adware** | Displays unwanted ads | Slows down system |
| **Rootkit** | Hides deep in system | Gives attacker admin control |
| **Keylogger** | Records keystrokes | Captures passwords, messages |
| **Botnet** | Network of infected computers | Used for DDoS attacks |
| **Fileless Malware** | Lives in RAM only | No files to detect |
#### 1. Virus
```
How it works:
User downloads infected file → Opens file → Virus activates → Infects other files

Example: ILOVEYOU virus (2000) - Caused $10 billion damage
```

#### 2. Worm
```
How it works:
Enters via vulnerability → Self-replicates → Spreads across network automatically

Example: WannaCry (2017) - Infected 200,000+ computers in 150 countries
```

#### 3. Trojan Horse
```
How it works:
Appears as legitimate software → User installs it → Creates backdoor → Attacker gains access

Example: Emotet - Banking trojan stealing financial data
```

#### 4. Ransomware
```
How it works:
Infects system → Encrypts all files → Displays ransom note → Demands cryptocurrency

Example: LockBit, REvil, Conti
```

#### 5. Spyware
```
How it works:
Installed secretly → Monitors all activity → Sends data to attacker

Example: Pegasus spyware - Used to spy on journalists
```

### How malware spread
```
1. Email Attachments (Phishing)
2. Malicious Downloads
3. Infected USB Drives
4. Compromised Websites
5. Software Vulnerabilities
6. Social Engineering
7. Drive-by Downloads
8. Peer-to-Peer Networks
```

### Basic Prevention
- Keep software updated
- Use antivirus software
- Don't click unknown links
- Avoid suspicious downloads
- Use strong passwords
- Regular backups


#  Antivirus Protection
Antivirus is software designed to detect, prevent, and remove malicious software (malware) from computers and networks.

Also known as:
- Anti-malware
- Endpoint Protection
- Security Software

### How Antivirus Works
#### Detection Methods

| Method | Description | Pros | Cons |
| :--- | :--- | :--- | :--- |
| **Signature-Based** | Matches known malware patterns | Fast, accurate for known threats | Cannot detect new/unknown malware |
| **Heuristic-Based** | Analyzes code behavior patterns | Can detect new variants | More false positives |
| **Behavioral-Based** | Monitors real-time activity | Detects zero-day attacks | Resource intensive |
| **Sandboxing** | Runs suspicious files in isolated environment | Safe testing | Slower detection |
| **Machine Learning** | AI learns malware patterns | Adapts to new threats | Requires training data |
| **Cloud-Based** | Uses cloud database for detection | Always updated | Needs internet connection |
### Antivirus Scanning Process

Step 1: File Access/Download
         ↓
Step 2: Antivirus Intercepts
         ↓
Step 3: Scan Against Database
         ↓
Step 4: Behavior Analysis
         ↓
Step 5: Decision
         ↓
    ┌────┴────┐
    ↓         ↓
  CLEAN    THREAT
    ↓         ↓
  Allow    Quarantine/Delete


### Types of Antivirus Scans

| Scan Type | Description | When to Use |
| :--- | :--- | :--- |
| **Quick Scan** | Scans common locations only | Daily/Regular use |
| **Full Scan** | Scans entire system | Weekly/Monthly |
| **Custom Scan** | Scans selected folders | Specific suspicion |
| **Real-time Scan** | Continuous monitoring | Always ON |
| **Boot-time Scan** | Scans before OS loads | Deep rootkit removal |

### Antivirus Limitations
Cannot detect all zero-day threats
Signature updates may lag behind new malware
Can slow down system performance
False positives possible
Cannot protect against social engineering
May conflict with other security software
Not effective against advanced persistent threats (APTs)

# Internet Security Tips to Know
## Password Security
```
✅ DO:
─────────────────────────────────
• Minimum 12-16 characters
• Mix uppercase + lowercase
• Include numbers
• Include special characters (!@#$%^&*)
• Use passphrases (e.g., "Coffee$Morning@2024!")
• Unique password for each account

❌ DON'T:
─────────────────────────────────
• Use personal info (name, birthday)
• Use dictionary words
• Use sequential numbers (123456)
• Use keyboard patterns (qwerty)
• Reuse passwords across sites
• Share passwords with anyone
```

## Social Engineering Awareness
```
COMMON SOCIAL ENGINEERING TACTICS
═════════════════════════════════════════

1. PRETEXTING
   → Attacker creates fake scenario
   → Example: "IT support" asking for password

2. BAITING
   → Offering something enticing
   → Example: Free USB drive with malware

3. QUID PRO QUO
   → Offering service for information
   → Example: "Free tech support" for access

4. TAILGATING
   → Following authorized person into building
   → Example: "I forgot my badge"

5. VISHING
   → Voice phishing via phone calls
   → Example: Fake bank call

DEFENSE: When in doubt, VERIFY through official channels!
```
