Programming is giving instructions to computers in a language they understand.

```
Human Thought → Programming Language → Machine Code → Computer Action
```

## Language Hierarchy
```
┌─────────────────────────────────────────────────┐
│  HIGH-LEVEL    │ Python, Java, JavaScript       │
│  (Human-like)  │ Easy to read and write         │
├─────────────────────────────────────────────────┤
│  MIDDLE-LEVEL  │ C, Go, Rust                    │
│  (Balanced)    │ Balance of control & ease      │
├─────────────────────────────────────────────────┤
│  LOW-LEVEL     │ Assembly, Machine Code         │
│  (Machine-like)│ Direct hardware control        │
└─────────────────────────────────────────────────┘
```
