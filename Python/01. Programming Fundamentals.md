Programming is the process of giving instructions to a computer to perform specific tasks. These instructions are written in a language that computers can understand and execute.

```
┌─────────────────────────────────────────────────────────────┐
│                    PROGRAMMING FLOW                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   Human Idea → Programming Language → Machine Code → Result │
│                                                             │
│   "Add 2+2"  →    print(2+2)      →   01010110   →   4     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

│                           ↑                                      │

## 2. Compiler & Interpreter

### Compiler
A compiler translates the entire source code into machine code at once, before the program runs.
```
┌─────────────────────────────────────────────────────────────────┐
│                      COMPILER PROCESS                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   Source Code          Compiler           Executable File        │
│   (program.c)    →    [Translate]    →    (program.exe)         │
│                        All at Once                               │
│                                                                  │
│   ┌──────────┐        ┌──────────┐        ┌──────────┐          │
│   │ int x=5; │   →    │          │   →    │ 01010110 │          │
│   │ int y=3; │        │ COMPILER │        │ 11001010 │          │
│   │ print(x) │        │          │        │ 10101011 │          │
│   └──────────┘        └──────────┘        └──────────┘          │
│                                                                  │
│   Time: Compile Once, Run Many Times                            │
│   Languages: C, C++, Go, Rust, Java (partially)                 │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**Characteristics of Compiler:**

- Translates entire code before execution
- Creates an executable file
- Shows all errors after compilation
- Faster execution (after compilation)
- Takes more memory during compilation
### Interpreter
An interpreter translates and executes the source code line by line, during runtime.
```
┌─────────────────────────────────────────────────────────────────┐
│                     INTERPRETER PROCESS                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   Source Code         Interpreter         Output                 │
│   (program.py)   →   [Line by Line]  →   (result)               │
│                                                                  │
│   ┌──────────┐        ┌──────────┐        ┌──────────┐          │
│   │ x = 5    │  →  1  │          │   →    │ (stored) │          │
│   │ y = 3    │  →  2  │INTERPRETER│   →    │ (stored) │          │
│   │ print(x) │  →  3  │          │   →    │ 5        │          │
│   └──────────┘        └──────────┘        └──────────┘          │
│                                                                  │
│   Time: Translates Every Time Program Runs                      │
│   Languages: Python, JavaScript, Ruby, PHP                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**Characteristics of Interpreter:**

- Translates code line by line
- No separate executable file created
- Stops at first error encountered
- Slower execution (translates every time)
- Easier debugging (immediate feedback)

```
┌─────────────────────────────────────────────────────────────────┐
│                 COMPILER vs INTERPRETER                          │
├──────────────────────┬──────────────────────────────────────────┤
│       COMPILER       │            INTERPRETER                    │
├──────────────────────┼──────────────────────────────────────────┤
│ Translates all at    │ Translates line by line                  │
│ once                 │                                          │
├──────────────────────┼──────────────────────────────────────────┤
│ Creates executable   │ No executable file                       │
│ file                 │                                          │
├──────────────────────┼──────────────────────────────────────────┤
│ Shows all errors     │ Stops at first error                     │
│ after compilation    │                                          │
├──────────────────────┼──────────────────────────────────────────┤
│ Faster execution     │ Slower execution                         │
├──────────────────────┼──────────────────────────────────────────┤
│ C, C++, Go, Rust     │ Python, JavaScript, Ruby                 │
└──────────────────────┴──────────────────────────────────────────┘
```
