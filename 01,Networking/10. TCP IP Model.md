# TCP/IP Model
The TCP/IP Model (Transmission Control Protocol/Internet Protocol) is a practical, four-layer networking framework that forms the foundation of the modern internet.

```
┌─────────────────────────────────────┐
│         APPLICATION LAYER           │  Layer 4
├─────────────────────────────────────┤
│          TRANSPORT LAYER            │  Layer 3
├─────────────────────────────────────┤
│           INTERNET LAYER            │  Layer 2
├─────────────────────────────────────┤
│        NETWORK ACCESS LAYER         │  Layer 1
└─────────────────────────────────────┘
```
