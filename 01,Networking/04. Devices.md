# 1.Repeater
- **Purpose:** Regenerates/boosts signal to extend distance.
- **Layer:** OSI Layer 1 (Physical)
- **Security view:** No filtering, no intelligence.
# 2. HUB (Layer 1 - Physical Layer)
A Hub is a basic networking device that connects multiple Ethernet devices together, making them act as a single network segment.

- **Purpose:** Sends data to ALL ports (broadcasts everything).
- **Layer:** OSI Layer 1 (Physical)
- **Key point:** One collision domain.
- **Security view:** Very insecure, easy sniffing.

## Working:
- **Broadcasting Device:** When a hub receives data on one port, it floods/broadcasts it to ALL other ports
- **No Intelligence:** Cannot read MAC addresses or make forwarding decisions
- **Half-Duplex:** Only one device can transmit at a time
- **Collision Domain:** All connected devices share ONE collision domain


# 3. BRIDGE

A bridge is a Layer 2 (Data Link Layer) device that connects two or more network segments and filters traffic based on MAC addresses.

- **Purpose:** Connects two LAN segments and filters using MAC.
- **Layer:** OSI Layer 2 (Data Link)
- **Key point:** Reduces collisions.
- **Security view:** Better than hub, but still limited.

# 4. SWITCH (Layer 2 - Data Link Layer)
- A Switch is essentially a "Multi-port Bridge." It creates a dedicated path between the sender and receiver.
- A Switch is an advanced multi-port bridge that operates at Layer 2 (or Layer 3 for multilayer switches). It's the backbone of modern LANs, providing dedicated bandwidth to each port.

- **Purpose:** Sends frames only to the correct port using MAC table.
- **Layer:** OSI Layer 2 (Data Link)
- **Key points:**
  - Uses MAC address table (CAM table)
  - Each port is separate collision domain

#  ==5. ROUTER (Layer 3 Device)==
- A Router is a Layer 3 device that interconnects different networks and makes forwarding decisions based on IP addresses. It's the fundamental device for internetworking and WAN connectivity.
- A Router is a Layer 3 device that forwards packets between DIFFERENT networks using IP addresses and routing tables.

- **Purpose:** Connects different networks and routes using IP.
- **Layer:** OSI Layer 3 (Network)
- **Key points:**
  - Uses routing table
  - Breaks broadcast domains


A Router is a network device operating at the Network Layer (Layer 3) that performs packet switching based on logical addressing (IP), maintains routing information bases (RIB), and executes routing protocols to determine optimal paths for data transmission across interconnected networks.
