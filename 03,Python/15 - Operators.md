## 1. Arithmetic Operators

```
┌──────────┬─────────────────┬─────────────────┐
│ Operator │ Name            │ Example         │
├──────────┼─────────────────┼─────────────────┤
│    +     │ Addition        │ 5 + 3 = 8       │
│    -     │ Subtraction     │ 5 - 3 = 2       │
│    *     │ Multiplication  │ 5 * 3 = 15      │
│    /     │ Division        │ 5 / 2 = 2.5     │
│    //    │ Floor Division  │ 5 // 2 = 2      │
│    %     │ Modulus         │ 5 % 2 = 1       │
│    **    │ Exponent        │ 5 ** 2 = 25     │
└──────────┴─────────────────┴─────────────────┘
```
